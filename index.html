JS Book Various JS applications

× Welcome to the JS Book scaffold. You can create various JS programs and showcase each app on different pages. Each page should feature a program using JS and/or jQuery that you can interact with. Build this site up so you can use it for your portfolio. Feel free to explore and experiment!

PROJECTS:

C/F Temperature Converter
  • Should take 2 inputs (C/F)
  • Calculate C to F
  • Calculate F to C
Gratuity Calculator
  • Should take 2 inputs (Total / Tip %)
  • Should be able to choose gratuity rate
  • "You should tip $___ on a bill of $___. "
Lottery Winner
  • Should take in input of names
  • Randomize a winner
  • Animate the page upon picking winner
Magic 8 Ball
  • Ask it a question
  • Randomize an output from an array of predefined answers
  • Create an 8 ball in CSS / find an image
  • Animate a fade in effect on the image
